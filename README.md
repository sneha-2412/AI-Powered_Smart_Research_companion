# AI-Powered Smart Research Companion

## 📌 Overview
AI-Powered Smart Research Companion is a browser-integrated web application that enhances online research productivity by generating concise, AI-driven summaries from selected web content.

It combines a Chrome Extension frontend with a Spring Boot backend, enabling users to highlight text on any webpage and instantly receive meaningful summaries powered by AI.

---

## ✨ Key Features
- 🔍 Instant Text Summarization – Generate concise summaries from selected content  
- 🌐 Chrome Extension Integration – Seamless in-browser interaction  
- 🧠 AI-Powered Processing – Uses Google Gemini API for intelligent summarization  
- 📝 Persistent Notes – Save and manage notes using browser local storage  
- 🔗 RESTful Architecture – Efficient communication between frontend and backend  

---

### Storage:
- Browser Local Storage  

---

## ⚙️ System Workflow
1. User selects text on any webpage  
2. Chrome Extension captures the selected text  
3. Request is sent to the backend via REST API  
4. Spring Boot backend processes the request  
5. Gemini API generates a summary  
6. Summary is returned and displayed in the extension  
7. User can save notes locally  

---

## 🧩 Architecture
- Chrome Extension Layer → Handles UI and user interaction  
- Backend Layer (Spring Boot) → API handling and AI integration  
- AI Layer (Gemini API) → Generates summaries  
- Storage Layer → Local storage for notes
